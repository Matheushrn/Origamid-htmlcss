Top,Right,Bottom,Left

Quando são adicionadas as propriedades Margin,Padding e Border automaticamente foram adicionados padding e margin para todos os cantos (lados) da caixa mas é possivel controlar e escolher o tamanho que queremos

Sequencia

Para colocar todos os lados em uma linha só é feito o seguinte

Div { padding:20px 40px; margin: 30px 100px 60px 200px } no jeito que está no margin seria Margin 30 Top 100 Right 60 Bottom 200 Left Isso é sempre lido no sentido horário agora quando se omite no caso do padding se le da seguinte forma Padding 20 Top/Bottom 40 Right/Left
