Grid Column

Propriedades

  • grid-column

    • Posiciona o item na coluna desejada.

  • 1 / 3

    • Da linha 1 até a linha 3.

  • 1 / -1

    • Da linha 1 até a última linha.

  • span 2

    • Expandir por 2 colunas.

Grid Column

Grid Template Rows

  • grid-template-rows

    • Define o tamanho das linhas.

  • grid-auto-rows

    • Linhas são adicionadas automaticamente com o valor de auto.

  • grid-row: 2

    • Define a linha do item, funciona da mesma forma que o grid-column.
